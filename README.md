# LSCHWCP_202391

This repository exists to address an error in the code link provided in the final online version of our [Nature Biotechnology paper](https://www.nature.com/articles/s41587-025-02614-y).

The link printed in the journal incorrectly points to this repository (`LSCHWCP_202391`) (the trailing "91" corresponds to a reference in the manuscript, which appears to have been inadvertently fused with the repository name during the final production process). This repository is **not** the official codebase and does not contain the project code.

The correct and official repository for this work is:

👉 https://github.com/pachterlab/LSCHWCP_2023

Please refer there for all code, documentation, and updates.

Thanks to [@iwangtoknow](https://github.com/iwangtoknow) for bringing this issue to our attention.
